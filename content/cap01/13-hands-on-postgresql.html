14  Hands-on: PostgreSQL na Prática

Autor

Douglas Braga

Nesta seção vamos instalar o PostgreSQL, criar o banco de dados universitário da UnDF e popular as tabelas com os dados que vimos ao longo do capítulo.

14.1 Instalação do PostgreSQL

  • Linux (Ubuntu/Debian)
  • macOS
  • Windows
sudo apt update
sudo apt install -y postgresql postgresql-contrib

# Iniciar e habilitar o serviço
sudo systemctl start postgresql
sudo systemctl enable postgresql

# Verificar se está rodando
sudo systemctl status postgresql

Após a instalação, acesse o console interativo:

sudo -u postgres psql

Com Homebrew instalado:

brew install postgresql@16

# Iniciar o serviço
brew services start postgresql@16

# Adicionar ao PATH (cole no ~/.zshrc ou ~/.bash_profile)
echo 'export PATH="/opt/homebrew/opt/postgresql@16/bin:$PATH"' >> ~/.zshrc
source ~/.zshrc

# Acessar o console
psql postgres

Alternativa: o app Postgres.app oferece instalação com um clique e não requer Homebrew.

  1. Acesse postgresql.org/download/windows e baixe o instalador EDB.
  2. Execute o instalador e siga o assistente. Anote a senha definida para o usuário postgres.
  3. O pgAdmin 4 (interface gráfica) é instalado junto e pode ser usado no lugar do terminal.

Para usar o terminal psql, abra o SQL Shell (psql) pelo Menu Iniciar e preencha as informações solicitadas (host: localhost, porta: 5432, usuário: postgres).

Alternativa: Caso não possua privilégios de administrador, siga essas instruções.

14.2 Criando o Banco de Dados

Conecte-se ao console do PostgreSQL e crie o banco:

CREATE DATABASE undf;

Em seguida, conecte-se a ele:

\c undf
Nota

No pgAdmin (Windows/Mac), clique com o botão direito em Databases → Create → Database, informe o nome undf e clique em Save. Depois abra o Query Tool para executar os comandos SQL.

14.3 Criando as Tabelas

As tabelas devem ser criadas na ordem correta: primeiro centro, pois as demais referenciam ela por chave estrangeira.

14.3.1 Tabela centro

CREATE TABLE centro (
    nome_centro VARCHAR(10)   PRIMARY KEY,
    predio      VARCHAR(50)   NOT NULL,
    orcamento   NUMERIC(10,2) NOT NULL
);

14.3.2 Tabela professor

CREATE TABLE professor (
    id          CHAR(5)      PRIMARY KEY,
    nome        VARCHAR(50)  NOT NULL,
    nome_centro VARCHAR(10)  REFERENCES centro(nome_centro),
    salario     NUMERIC(8,2) NOT NULL
);

14.3.3 Tabela disciplina

CREATE TABLE disciplina (
    id_disciplina VARCHAR(10)  PRIMARY KEY,
    titulo        VARCHAR(100) NOT NULL,
    nome_centro   VARCHAR(10)  REFERENCES centro(nome_centro),
    carga_horaria INTEGER      NOT NULL CHECK (carga_horaria > 0)
);

14.3.4 Tabela aluno

CREATE TABLE aluno (
    id          CHAR(5)     PRIMARY KEY,
    nome        VARCHAR(50) NOT NULL,
    nome_centro VARCHAR(10) REFERENCES centro(nome_centro),
    carga_total INTEGER     NOT NULL DEFAULT 0
);

Para confirmar que as tabelas foram criadas:

\dt

14.5.2 Professores do COETI com salário acima de 70 000

SELECT nome, salario
FROM professor
WHERE nome_centro = 'COETI'
  AND salario > 70000;

14.5.3 Carga horária total por centro

SELECT nome_centro, SUM(carga_horaria) AS total_horas
FROM disciplina
GROUP BY nome_centro
ORDER BY total_horas DESC;

14.5.4 Alunos com mais de 500 h acumuladas

SELECT nome, nome_centro, carga_total
FROM aluno

DicaDesfazendo tudo (reset)

Para remover todas as tabelas e começar do zero:

DROP TABLE aluno, disciplina, professor, centro;

A ordem importa: tabelas com chaves estrangeiras devem ser removidas antes da referenciada, ou use CASCADE:

DROP TABLE centro CASCADE;
